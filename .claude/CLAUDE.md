# 番茄钟 (Pomodoro Clock)

Pure frontend pomodoro clock application. HTML+CSS+原生 JavaScript, no frameworks.

## Commands
- `npm test` -- run Jest tests
- `npm run test:watch` -- run Jest in watch mode

## Architecture
- `src/` -- all source code (app.js, timer.js, state.js, storage.js, ui.js, stats.js, index.html, styles.css)
- `tests/` -- Jest unit tests
- Serving: open `src/index.html` in browser (no build step)
